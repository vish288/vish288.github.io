## Stack

React 19 · TypeScript 6 · Vite 8 · Tailwind CSS 4 · shadcn/ui + Radix · Vitest 4

## Pages

| Route           | Description                                               |
| --------------- | --------------------------------------------------------- |
| `/`             | About — profile, skills word cloud, highlight cards       |
| `/repositories` | GitHub repos with filters, sorting, and infinite scroll   |
| `/mcp-install`  | MCP server installation gateway (GitLab, Atlassian, Coda) |

## Quick start

```bash
git clone https://github.com/vish288/vish288.github.io.git
cd vish288.github.io
pnpm install
pnpm dev
```

Requires Node >= 24 and pnpm >= 10. See [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md) for the full script reference, release flow, and contribution notes.
