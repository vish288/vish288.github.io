**[vish288.github.io](https://vish288.github.io)**

## Pages

| Route           | Description                                                                      |
| --------------- | -------------------------------------------------------------------------------- |
| `/`             | About — profile, skills word cloud, highlight cards                              |
| `/repositories` | GitHub repos with filters, sorting, and infinite scroll                          |
| `/mcp-install`  | MCP server installation gateway (GitLab, Atlassian, Coda) with deep-link support |
| `/gratitude`    | Contact form with captcha and trust signals                                      |

## Quick Start

Requires **Node >= 20** and **pnpm >= 8**.

```bash
git clone https://github.com/vish288/vish288.github.io.git
cd vish288.github.io
pnpm install
pnpm run dev
```

See **[docs/DEVELOPMENT.md](docs/DEVELOPMENT.md)** for scripts, testing, and contributing guidelines.
